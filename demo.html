ThemeStudio

A pure CSS theming library for semantic HTML. Drop in one link tag, write clean HTML, and your page is styled. Switch themes by changing a single word. No classes, no configuration, no framework required.

Code

Use inline code for snippets inside prose. Use Ctrl + V for keyboard shortcuts. 

<!-- ThemeStudio setup -->
<link rel="stylesheet" href="themestudio/reset.css">
<link rel="stylesheet" href="themestudio/themes/minimal/basic.css">

<!-- Optional utilities -->
<link rel="stylesheet" href="themestudio/utilities.css">

<!-- Write semantic HTML, everything is styled -->
<button>Already styled</button>
<nav><a href="/">No classes needed</a></nav>

Details & Summary

How do I switch themes?

Change the theme filename in your link tag. Replace basic.css with any other theme and the entire page restyles automatically.

Does this work with React or Next.js?

Yes. ThemeStudio is pure CSS with no JavaScript dependency. Import it globally and write semantic JSX — it just works.

What does minimal tier mean?

Minimal themes provide a clean canvas with sensible defaults. The library stays out of your way while still making your page look intentional and readable.
